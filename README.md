# CIFAR-10 CNN Image Classification

A Convolutional Neural Network built with TensorFlow/Keras to classify images across 10 object categories from the CIFAR-10 dataset.

## Model Performance
- **Test Accuracy:** 68.65%
- **Test Loss:** 0.9143
- **Architecture:** 3 convolutional layers, max pooling, dropout regularization

## Project Structure
- `cifar10_cnn.ipynb` - Complete training and evaluation notebook
- `my_cnn_model.h5` - Trained model weights

## Technologies Used
- TensorFlow/Keras
- Python, NumPy, Matplotlib
- scikit-learn
